# 🚀 Dodge The Blocks

A mobile-friendly arcade dodge game built using HTML, CSS, and JavaScript.

## 🎮 Game Description

Dodge The Blocks is a simple arcade-style game where the player must move left and right to avoid falling enemy blocks. The longer the player survives, the higher the score becomes.

## ✨ Features

- Player movement system
- Keyboard controls
- Mobile touch controls
- Falling enemy blocks
- Multiple enemy types
- Collision detection
- Score system
- Difficulty increase
- Start screen
- Game Over screen
- Background music
- Game over sound effect
- Particle explosion effect
- Responsive mobile design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🎮 Controls

### Desktop
- Left Arrow key → Move left
- Right Arrow key → Move right

### Mobile
- Tap ⬅️ button → Move left
- Tap ➡️ button → Move right

## 🚀 How to Play

1. Click **Start Game**.
2. Move the player left or right.
3. Avoid the falling blocks.
4. Try to get the highest score.
